Asset Management Module – User Manual

NOTE:
This section explains how to use the Asset Management module. Follow the steps carefully while entering or importing asset data.
# Tab Purpose Instructions
1 Details View and manage all asset records • View asset list in table format
• Use Search to find assets quickly
• Use column sorting arrows to sort data
• Click Asset ID to view asset details
• Click Edit to modify record
• Use checkbox to select records and delete
2 New Record Add new asset information • Open New Record tab
• Enter asset information such as asset code, type, department, vendor etc.
• Click Add More to create multiple rows
• Click Save All to store records
3 Bulk Import Upload multiple asset records at once • Upload supported file formats:
CSV / Excel / DOCX / TXT
• File must follow system column format
• Invalid rows will show error messages
4 Bulk Export Download asset data • Use filters or search if required
• Click export buttons in toolbar
• Supported formats:
DOC / PDF / Excel / CSV
5 Status Timeline View asset lifecycle history • Click asset status in Details table
• Timeline shows:
Asset creation date
Installation date
AMC period
Last service date
Current status
6 Help Guidelines for using this module • Read instructions before using the system
• Follow validation messages
• Use this page for troubleshooting
